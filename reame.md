<h1>Книги</h1>

_Финальный проект студента группы Led Zeppelin Ждановских Александра_

В проекте используются два пользователя: __admin__ и __user__ с такими же паролями.

В проекте реализовано:
- база данных, связи один-ко-многим, многие-ко-многим
- CRUD по книгам в двух вариантах: web (http://localhost:8080/final-project) и REST (http://localhost:8080/final-project/api)
- Spring Security, базовая аутентификация. Пользователь из группы ADMIN может выполнять все действия, пользователь из группы USER - только чтение.
- глобальный обработчик ошибок.
- инициализация базы через FlyWay
- maven ...
